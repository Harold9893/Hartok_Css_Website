# HARTOK - Web Solutions

## Descripción

HARTOK Web usando Css y Spline

## Características

-  Diseño moderno y responsivo

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos y animaciones
- **JavaScript** - Interactividad (Spline Viewer)
- **Spline** - Visualizador 3D interactivo

## Estructura del Proyecto

```
build_stunning/
├── index.html      # Página principal
├── styles.css      # Estilos del proyecto
├── gradient.png    # Imagen de gradiente de fondo
└── README.md       # Este archivo
```

## Uso

1. Abre `index.html` en tu navegador web
2. Explora la interfaz y navega a través de los menús
3. Haz clic en los botones para acceder a documentación o comenzar

## Secciones Principales

- **Header** - Navegación principal con logo HARTOK
- **Contenido Principal** - Presentación de la solución con modelo 3D
- **Navegación** - Links a Compañía, Nosotros, Recursos y Docs
- **CTA** - Botones de "Documentación" e "Empieza"

## Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet (para cargar el modelo 3D de Spline)
